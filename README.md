# 🎎프로젝트 소개

명예나 사회적 관계에 대한 일본 문화의 세부적인 부분 중 하나로,
일본 사회에서는 결혼식에 대한 참여 여부나 축의금에 관한 문제를
공개적으로 표현하기 어려워합니다.

그렇기에 그들의 프라이빗을 지키며 동시에 즐거움을 주어서
서로의 부담감을 덜고 나아가 서로의 인연을 지키는것에 일조하기위한
프로젝트입니다.
